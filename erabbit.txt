  <h1>Project: Calculator Euphoria</h1>

  A free collection of online calculators for math, finance, health and everyday life

  <h2>Stack</h2>

  <ul>
    <li>- HTML/CSS/JS: Plain, no framework or build step</li>
    <li>- DevOps: Static files on GitHub Pages, custom domain (calculatoreuphoria.com)</li>
  </ul>

  <h2>Objective</h2>

  To bundle a dozen self-contained calculators — mortgage, loan, compound
  interest, tip, BMI, calorie, percentage, GPA, age, discount, scientific,
  and unit conversion — behind a single searchable homepage, each one
  computing entirely in the browser.

  <h2>Benefits</h2>

  <ul>
    <li>- Every calculation runs client-side; nothing typed is ever sent to a server</li>
    <li>- Shared design system means adding a new calculator is mostly copy-and-adapt</li>
    <li>- Hero search and category filters make finding the right calculator fast</li>
    <li>- No backend or database, so the whole site is free to host</li>
  </ul>

  <h2>Drawbacks</h2>

  <ul>
    <li>- No templating layer, so shared header/footer chrome is duplicated across every page</li>
  </ul>

  <h2>Easter Eggs</h2>

  A tiny rabbit sits tucked into the very bottom-right corner of the page,
  a nod to eRabb.it (erabb.it) — the site this one was built alongside.
